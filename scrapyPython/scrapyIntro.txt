Scrapy intro:(These commands to be run into scrapy shell)

To run scrapy shell execute in Anaconda prompt:-
$ scrapy shell

1. To fetch the url use

	fetch('yourURL here')
ex>	fetch('https://www.reddit.com/r/funny/')

2. To check the url use

	response.url

3. to extract out the contents using the CSS or any other webpage prop use the following syntax

	response.css(' `pageTag`.`classTitle` ')
ex > 	response.css('a.s1cbwutf-1')

4. To extract the useful content ou of it use the following command

	response.css(' `pageTag`.`classTitle` ').extract()
ex > 	response.css('a.s1cbwutf-1').extract() 	
5. To extract out only specific attributes from the url 

	response.css(' `pageTag`.`classTitle`::attr(href)').extract	
ex >	response.css('a.s1cbwutf-1.attr(href)').extract
